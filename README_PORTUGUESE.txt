Projeto_Login
Projeto feito para praticar meus conhecimentos em Flask e TailwindCSS

Para rodar o projeto na sua maquina:

- Tenha o framework Flask instalado
- Clone o repositório
- Execute o arquivo "projeto.py"

Caminhos do projeto:

http://127.0.0.1:5000/login - "/login" é a Tela inicial do projeto.
http://127.0.0.1:5000/mostrar_lista - "/mostrar_lista" é o lugar que aparece ao fazer login, mostra os nomes dos usuários já criados. 
http://127.0.0.1:5000/criar - "/criar" é onde cria um novo usuário. 

É necessário estar logado para acessar a página "/mostrar_lista"

Inicialmente não terá usuários, então crie eles e experimente como o sistema funciona. Caso encontre algum problema, me avise pelo meu e-mail se possível.

Nota: Nenhum desses usuários são reais, eles não são salvos em lugar algum e somem quando o arquivo "projeto.py" para de ser executado.
